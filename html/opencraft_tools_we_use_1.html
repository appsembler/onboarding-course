There're multiple tools we use. Please read our docs.

You will be given access to them - unless specified, you won't have anything to do to obtain it.

Be sure to read the introduction to Ocim (OpenCraft Instance Manager), which is the custom tool that we use to deploy instances. Learning about it will allow you to understand much about our infrastructure, so it's recommended that you get an account and try to use it soon.

We also have a link shortener that you can install by adding 149.202.185.230 o to your /etc/hosts, and it will translate links to tasks (e.g. https://o/OC-515), and to other pages (https://o/board, https://o/agenda, https://o/timesheet, https://o/contact, etc.) See full list at https://o/links
