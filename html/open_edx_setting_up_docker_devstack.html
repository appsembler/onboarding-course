Setting up a docker devstack is the easiest and faster way of having a fully feature instance of Open EdX working and ready for development on your local machine. This talk (slides), given by our own team members, is an optional introduction to the docker devstack.

The following instructions will help you set up the docker devstack:

  1. Read about the Installation Prerequisites and make sure you have all the tools installed on your OS.
  2. Create a dedicated folder for each devstack (eg:. ~/hawthorn-devstack/).
  3. Clone the devstack configuration repo into it using git clone https://github.com/edx/devstack
  4. Enter the folder where you just cloned the repo and set up a Python virtualenv in order to install the devstack's requirements.
  5. Run make requirements to install all python prerequisites needed for provisioning.
  6. Follow the instructions on Installing Devstack of EdX documentation.
