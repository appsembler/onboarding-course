The Open edX project is a web-based platform for creating, delivering, and analyzing online courses. You can get to know the Open EdX architecture a little better here, take a look before going forward on this chapter.

OpenEdX periodically releases stable versions of the platform called named releases. Since not every instance/client uses the same release, it's useful to know how the releases work and what's the current one. Take a moment to read Open EdX Named Releases before setting up your development environment.

Most of the codebase is coded in Python 2.7, with the exception of the forum service, which is coded in Ruby.
