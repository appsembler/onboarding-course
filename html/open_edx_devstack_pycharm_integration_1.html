Open edX has extensive documentation on integrating your favorite IDE with devstack.

See below for more details about specific IDEs:

  • PyCharm: Docker devstack, Vagrant devstack

    Note: there is currently a problem with hanging debugger with docker-compose in PyCharm. The hack for this is to create separate config for running debug with additional option --noreload. Please check this YouTrack ticket to see the issue status.

  • VS Code: Docker Devstack.
  • Vim: Any devstack.
