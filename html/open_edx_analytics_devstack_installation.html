Analytics developer stack

The Open edX Analytics Developer stack, known as the Analytics Devstack, is a modified version of the Open edX Developer Stack. It provides all of the services and tools needed to modify the Open edX Analytics Pipeline, Data API, and Insights projects.

OpenCraft's clients wouldn't be using the Analytics Developer Stack (unless they wanted to make changes to any of edX's analytics repos themselves). However, OpenCraft hosts production deployments of edX Insights for a few clients. So if you're about to work with one of these clients you might need to install the Analytics Developer Stack.
