In some tasks, you'll need to add on change functionality on Ocim, and for that you will need to set up a Ocim devstack.

In order to make the installation easier, we'll provision a vagrant machine containing all the tools needed to run a fully featured Ocim instance locally. To do that, use the following instructions:

1. Install VirtualBox and Vagrant on your machine.

2. Clone the ocim repo to a directory of your choosing with git@github.com:open-craft/opencraft.git

3. Follow the instructions under Vagrant install. After successfully completing these steps, you should have a functioning Ocim devstack.

4. In order to to run a Ocim server and provision/create Open EdX instances and Appserver, you will need to configure environment variables that give it access to our OpenStack account used for testing. You can use the credentials on this file for testing your changes to Ocim locally.

Don't forget to read the entire documentation on Ocim to make sure you know how to use it.
